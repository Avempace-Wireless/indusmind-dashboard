2rem;\n  }\n}\n```\n\n### Key Breakpoints\n```\nMobile:   < 768px    (default)\nTablet:   768px-1023px\nDesktop:  1024px-1279px\nLarge:    1280px+\n```\n\n## Color Palette System\n\n### Primary Colors\n```css\n--color-text-primary:    #111827  /* bodydark */\n--color-text-secondary:  #6b7280  /* bodydark2 */\n--color-text-tertiary:   #d1d5db  /* gray-300 */\n\n--color-bg-white:        #ffffff\n--color-bg-light:        #f9fafb  /* gray-50 */\n--color-bg-lighter:      #fafafa  /* very light */\n--color-bg-hover:        #f3f4f6  /* gray-100 */\n\n--color-border:          #e5e7eb  /* gray-200 */\n--color-border-light:    #f3f4f6  /* gray-100 */\n\n--color-accent:          #10b981  /* green */\n--color-accent-dark:     #059669  /* darker green */\n```\n\n## Common Customizations\n\n### Changing Theme Colors\n```css\n/* Update these values throughout: */\n#10b981  → Your primary color\n#059669  → Your darker primary\n#e5e7eb  → Your border color\n#111827  → Your text color\n```\n\n### Adjusting Spacing\n```css\n/* Section gaps */\ngap: 1.5rem;  → Modify for tighter/looser layout\n\n/* Panel padding */\npadding: 1.5rem;  → Adjust for different densities\n```\n\n### Chart Height\n```css\n/* Desktop default: */\n.energy-chart__container {\n  height: 450px;  /* Adjust as needed */\n}\n```\n\n### Calendar Day Size\n```css\n/* Day cell sizing: */\n.calendar-selector__day {\n  min-height: 48px;  /* Touch target size */\n  aspect-ratio: 1;   /* Keep square */\n}\n```\n\n## Performance Considerations\n\n### Animations\n```css\n/* Use GPU-accelerated properties: */\ntransform: translateY(-2px);  ✓ Good\nopacity: 0.5;                 ✓ Good\nbox-shadow: ...;              ⚠️ Moderate\n\n/* Avoid reflow-causing changes: */\nheight: ...;       ✗ Causes reflow\nwidth: ...;        ✗ Causes reflow\npadding: ...;      ✗ Causes reflow\n```\n\n### CSS Order\n1. Display properties\n2. Box model (padding, margin, border)\n3. Colors and backgrounds\n4. Typography\n5. Transforms and transitions\n\n## Browser Support\n\n```\nChrome:       Latest (full support)\nFirefox:      Latest (full support)\nSafari:       Latest (full support)\nEdge:         Latest (full support)\nIE11:         Not supported (CSS Grid, Flexbox)\n```\n\n## Development Workflow\n\n### Adding New Section\n1. Create BEM block name (e.g., `.new-section`)\n2. Define elements (e.g., `.new-section__header`)\n3. Add modifiers for states (e.g., `.new-section__item--active`)\n4. Apply responsive rules\n5. Test on all breakpoints\n\n### Modifying Existing Section\n1. Locate section in energy-dashboard.css\n2. Find relevant block/element/modifier\n3. Update styling\n4. Test responsive behavior\n5. Check for unintended side effects\n\n### Color Updates\n1. Search and replace throughout file\n2. Update color palette documentation\n3. Test on both light and dark backgrounds\n4. Verify accessibility contrast\n\n## Testing Checklist\n\n- [ ] Mobile layout (320px, 375px, 480px)\n- [ ] Tablet layout (768px, 1024px)\n- [ ] Desktop layout (1280px, 1440px, 1920px)\n- [ ] Touch interactions on mobile\n- [ ] Keyboard navigation\n- [ ] Hover states on desktop\n- [ ] Focus states for accessibility\n- [ ] Color contrast ratios (WCAG AA)\n- [ ] Performance (Lighthouse)\n- [ ] Dark mode appearance (if applicable)\n\n## Debugging Tips\n\n### Check Layout Issues\n```css\n/* Temporary debug border */\n* { border: 1px solid red; }\n\n/* Show grid lines */\nbody { background-image: \n  linear-gradient(rgba(0,0,0,0.1) 1px, transparent 1px),\n  linear-gradient(90deg, rgba(0,0,0,0.1) 1px, transparent 1px);\n  background-size: 1rem 1rem;\n}\n```\n\n### Element Inspection\n- Use Chrome DevTools Inspector\n- Check Computed Styles tab\n- Verify box model\n- Test responsive modes\n\n### Performance Profiling\n- Use Chrome DevTools Performance tab\n- Record animations\n- Check for layout thrashing\n- Optimize expensive selectors\n\n## Resources\n\n- [BEM Methodology](https://getbem.com/)\n- [CSS Grid](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Grid_Layout)\n- [Flexbox](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Flexible_Box_Layout)\n- [Responsive Design](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Responsive_Design)\n- [Accessibility](https://www.w3.org/WAI/fundamentals/)\n\n---\n\n**Last Updated**: 2026-01-09\n**Maintainer**: Design System Team\n**Status**: Production Ready ✅\n"
